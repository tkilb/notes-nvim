| Key        | Action                   |
| ---------- | ------------------------ |
| <leader>]  | Switch to next tab       |
| <leader>[  | Switch to prev tab       |
| :tabe      | new empty tab            |
| :tab split | split current tab        |
| :tabe %    | alias, split current tab |
| :tabc      | Close tabs               |
| :tabo      | Close all other tabs     |
