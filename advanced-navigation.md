## In Normal mode

| Key    | Action                                                                                    |
| ------ | ----------------------------------------------------------------------------------------- |
| 1<C-g> | Print cwd of open file buffer                                                             |
| <C-b>  | Half page up                                                                              |
| <C-d>  | Half page down                                                                            |
| <C-o>  | Go back                                                                                   |
| <C-i>  | Go forward                                                                                |
| :noh   | Clear '/' search                                                                          |
|        | https://dev.to/m4xshen/3-vim-commands-for-blazingly-fast-navigation-between-brackets-55kc |
